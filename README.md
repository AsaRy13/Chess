# Overview

This is supposed to be a simple chess program made in Kotlin. However, right now it doesn't do anything but show the board and the staring positions of the pieces. What I am trying to do with this project is learn Kotlin and its GUI capabilities, and I have learned a lot. What I plan next to do with this project is to finish piece functionality and make it an app.
